Download the files in this directory:  
One file is `uniprot_segments_observed.tsv`, which is a summary of the UniProt to PDBe residue level mapping (observed residues only).
```
wget ftp://ftp.ebi.ac.uk/pub/databases/msd/sifts/flatfiles/tsv/uniprot_segments_observed.tsv.gz
```
Anothe file is `All_species_train.csv`, PON-All predicts the tolerance (pathogenicity) of amino acid substitutions in multiple species.   
```
wget http://structure.bmc.lu.se/VariBench/EXTENSION/VariationTypeDatasets/SubstitutionsCodingRegion/TrainingDatasets/Dataset25/All_species_train.csv 
```
